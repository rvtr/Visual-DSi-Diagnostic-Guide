Visual DSi Fault Guide

Description

Unlaunch will not display anything with the SD card inserted, or it will display an "empty MBR" error.

There are a couple reasons this can happen:
- If unlaunch displays nothing or mentions MBR, the SD card may be corrupted or improperly formatted.
- If unlaunch mentions clusters being too large, the SD card has not been formatted with a 32KB cluster size.

Solution

You will need to back up and format your SD card. See the following guide for instructions: https://dsi.cfw.guide/sd-card-setup.html
