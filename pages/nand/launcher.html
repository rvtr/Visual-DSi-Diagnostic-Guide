Description

The DSi boots to two black screens with the text "Error: 1-2435-8325"

This happens when the DSi's launcher (home menu) is corrupted or missing.

Solution

Restore a NAND backup with ntrboot.

If you don't have a NAND backup, you can try reinstalling the launcher. Download the launcher TAD with NUSGet and then install it with TDT. You will still need ntrboot to boot TDT.
